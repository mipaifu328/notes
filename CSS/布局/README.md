## margin纵向重叠问题
- 相邻元素的margin-top和margin-bottom会发生重叠
- 空白内容的`<p></p>`会重叠

## margin负值问题
- margin-top 负值，元素向上移动
- margin-left 负值，元素向左移动
- margin-right 负值， *自身不受影响*，右侧元素向左移动
- margin-bottom 负值， *自身不受影响*，下方元素向上移动
